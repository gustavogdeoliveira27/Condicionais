# Condicionais
Exercícios de estruturas condicionais propostas na matéria de Raciocínio Algorítmico na faculdade PUCPR. A lista de exercícios utiliza a linguagem de programação Python, e foram desenvolvidos no editor de código-fonte VS Code. É formada por exercícios de condicionais simples e compostas, utilizando a entrada do usuário e if, elif e else para determinar sua saída.
